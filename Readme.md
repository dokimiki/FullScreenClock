FullScreenClock
-----

## Overview

時計です。

## Description

全画面表示の時計です。文字の色、フォント、大きさ 時計の更新速度などをカスタマイズすることができます。

## Usage

FullScreenClock.exeを実行してください。
EscキーかSpaceキーで設定モードに入れます。
終了は設定モードから行えます。
